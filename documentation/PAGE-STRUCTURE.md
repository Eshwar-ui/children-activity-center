# Page Structure

This project includes a root home page plus additional pages under `pages/`.

## Root entry

- `index.html`: primary landing page from the project root

## Home pages

- `pages/index.html`: alternate home page within the pages directory
- `pages/index-2.html`: second alternate home page

## Standard content pages

- `pages/about.html`: brand story, trust, and center overview
- `pages/services.html`: activity tracks and program details
- `pages/blog.html`: editorial and insight page
- `pages/contact.html`: contact form and location details

## Auth and utility pages

- `pages/login.html`: user sign-in
- `pages/signup.html`: account creation
- `pages/404.html`: error state page
- `pages/coming-soon.html`: placeholder launch page

## Dashboard pages

### User dashboard

- `pages/dashboard-user.html`: parent overview
- `pages/dashboard-user-children.html`: child profile view
- `pages/dashboard-user-calendar.html`: schedule view
- `pages/dashboard-user-billing.html`: billing view

### Admin dashboard

- `pages/dashboard-admin.html`: admin overview
- `pages/dashboard-admin-families.html`: family management view
- `pages/dashboard-admin-tracks.html`: track management view
- `pages/dashboard-admin-financials.html`: finance overview

## Shared asset structure

- `assets/css/style.css`: core design system and layout
- `assets/css/dark-mode.css`: dark mode overrides
- `assets/css/rtl.css`: right-to-left overrides
- `assets/css/tailwind-local.css`: local utility layer
- `assets/js/main.js`: shared interaction logic
- `assets/js/theme-handler.js`: theme and direction persistence
- `assets/js/dashboard.js`: dashboard UI helpers
- `assets/js/animations.js`: shared animation placeholder

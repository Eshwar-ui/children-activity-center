# BrightSteps HTML Template

BrightSteps is a children's activity center website template built with static HTML, shared CSS, and shared JavaScript. This README has been aligned with `updated_system_instructions.md` and with the current project contents.

## Project overview

- Primary root page: `index.html`
- Additional pages: `pages/`
- Shared assets: `assets/`
- Documentation: `documentation/`

The template includes:

- home pages
- standard marketing pages
- auth pages
- dashboard pages
- local brand assets
- local fonts
- dark mode support
- RTL support

## Local assets

- Fonts are stored in `assets/fonts/`
- Images are stored in `assets/images/hero/`, `assets/images/content/`, and `assets/images/ui/`
- Logo: `assets/images/ui/logo.svg`
- Favicon: `assets/images/ui/favicon.svg`
